# Responsive Portfolio Website

This is a simple and responsive portfolio web page built using HTML and CSS as part of a Web Development Internship task. The goal was to convert a desktop-only layout into a mobile-friendly design using media queries.

## Features
- Responsive navigation
- Mobile-optimized layout
- Project cards with modern UI
- Profile image and intro section

## Tools Used
- HTML5
- CSS3 (Flexbox, Media Queries)
- Chrome DevTools for testing

## How to Run
Clone the repository and open `index.html` in your browser.
