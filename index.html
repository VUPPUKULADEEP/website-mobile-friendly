old books management system

This Old Books Management System is a full-stack web application built with FastAPI (backend) and React (frontend), designed to manage and track used or second-hand books. The backend uses SQLAlchemy ORM with a SQLite3 database for efficient data modeling and storage. CRUD operations (Create, Read, Update, Delete) are exposed through RESTful APIs, ensuring a clean separation between frontend and backend logic. The frontend, built in React, offers a responsive and user-friendly interface for interacting with the system.
